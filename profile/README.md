## Imbricate CLI

Imbricate CLI is available through NPM for package management. See https://github.com/imbricate/Imbricate-CLI.

To install Imbricate CLI:

### Using Yarn

Yarn 1.x

```sh
yarn global add imbricate
```

Yarn 2.x

```sh
yarn dlx -p imbricate
```

### Using NPM

```sh
npm install -g imbricate
```
